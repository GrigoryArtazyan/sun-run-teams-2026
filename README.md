# sun-run-teams-2026
Data dashboard for the 2026 Vancouver Sun Run 10K team results.

## Data

Results are open, public data from Sportstats (Vancouver Sun Run):

- [10 km leaderboard](https://sportstats.one/event/vancouver-sun-run/leaderboard/146265)
- [2026 team results (published page)](https://cdn-1.sportstats.one/SunRun2026_Teams.htm)

## Quick start (Conda)

```bash
conda env create -f sunrun.yml
conda activate sunrun
streamlit run app.py
```
